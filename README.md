# Setup


Install the ZoKrates CLI:

```
curl -Ls https://scrypt.io/scripts/setup-zokrates.sh | sh
```

Setup and check the zkSNARK verifier:

```
sh ./scripts/setup.sh
```


Running `verifier.js`.

```
node --max-old-space-size=8192 verifier.js --run
```
